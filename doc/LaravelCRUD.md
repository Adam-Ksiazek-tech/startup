# Laravel CRUD


## Stawianie aplikacji

### Curl robi całą magię

curl -s https://laravel.build/startup-task | bash

### Uruchomienie dockera

```cd moj-projekt
./vendor/bin/sail up
```

### Uruchomienie migracji 

```
./vendor/bin/sail artisan migrate
```

### Parametry testowe dla bazy danych

host: 127.0.0.1
Port: 3306
User: sail
Password: password
Database: laravel
Driver: MySQL
